# Car Payment Calculator

Free, open-source car payment calculator. Calculate monthly auto loan payments with down payment, trade-in, and interest rate.

## [Car Payment Calculator](https://carpaymentcalculator.app/)

The full version includes:
- Credit score-based rate matching
- ZIP code sales tax lookup
- Reverse calculator (budget to max car price)
- Term comparison (24-84 months side by side)
- Full amortization schedule with charts

## Quick Calculator

A simplified version is available at: https://wsmhj.github.io/car-payment-calculator/

## Formula

M = P x r(1+r)^n / ((1+r)^n - 1)

Where:
- M = monthly payment
- P = principal (loan amount)
- r = monthly interest rate (annual rate / 12)
- n = number of payments (loan term in months)
